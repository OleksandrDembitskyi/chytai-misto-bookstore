# 📚 Читай-місто — Фронтенд (базова версія)

Це **базова робоча версія** фронтенду книгарні. Вона була створена на початковому етапі розробки для демонстрації інтерфейсу.

## ⚠️ Увага

- Ця гілка містить **тільки статичні файли** (HTML/CSS/JS).
- Для повноцінної роботи з API потрібен бекенд.
- **Фінальна стабільна версія** знаходиться в гілці `develop` (або `main`).

## 🚀 Запуск

Щоб переглянути статичну версію фронтенду, відкрийте файл `public/index.html` у браузері.

Для роботи з API використовуйте гілку `develop`.

## 🔗 Повна версія

- Онлайн сайт: `https://chytai-misto-bookstore.vercel.app`
- Гілка з повною версією: `develop`
