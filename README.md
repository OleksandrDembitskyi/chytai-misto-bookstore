# 📚 Читай-місто — Бекенд (базова версія)

Це **базова робоча версія** бекенду книгарні. Вона була створена на початковому етапі розробки для демонстрації основних можливостей API.

## ⚠️ Увага

- Ця гілка містить **тільки серверну частину** (API) без фронтенду.
- Тут можуть бути відсутні деякі виправлення та доробки, які були внесені пізніше.
- **Фінальна стабільна версія** знаходиться в гілці `develop` (або `main`).

## 🚀 Запуск (тільки API)

npm install
