# Читай-місто — Онлайн-книгарня

Веб-застосунок для книгарні з Node.js, MongoDB та адаптивним фронтендом.

## Запуск

1. Встанови залежності: `npm install`
2. Запусти сервер: `npm start`
3. Відкрий `index.html` або перейди на `http://localhost:3000`

## Технології

- Backend: Node.js + Express
- Database: MongoDB
- Frontend: HTML/CSS/JS
